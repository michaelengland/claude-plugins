# Claude Plugins

A marketplace of Claude Code plugins by Michael England.

## Installation

```
/plugin marketplace add michaelengland/claude-plugins
```

Then browse available plugins with `/plugin`.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [deckwright](https://github.com/michaelengland/deckwright) | Presentation workflow plugin: narrative, craft, design, present, and import |
